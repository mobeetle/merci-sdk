Getting Started

Follow these steps to set up the project and run your first example.

Prerequisites

  • Node.js (v18.x or newer recommended)

  • An active JetBrains AI Platform account and a JSON Web Token (JWT)

Installation & Setup

1. Clone the Repository

Get the latest version of the tutorials from GitHub.

git clone https://github.com/your-repo/merci-sdk-tutorials.git
cd merci-sdk-tutorials

2. Install Dependencies

This project has minimal dependencies, which can be installed via npm.

npm install

3. Configure Your Authentication Token

Create a new file in the project root named token.mjs and add the following content, replacing `"YOUR_JWT_HERE"` with your actual token.

// This file securely exports your authentication token.
// It is imported by all tutorial lessons.

export const token = "YOUR_JWT_HERE";
