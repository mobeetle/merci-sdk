Get Started View Tutorials

Focus on your logic. Let Merci do the rest.

The SDK's high-level agent (`.run()`) handles the entire multi-step tool-use process automatically, turning complex logic into just a few lines of code. 

import { MerciClient } from './merci.2.11.0.mjs';

// 1. Define your tools with simple objects
const weatherTool = {
  name: 'get_current_weather',
  description: 'Get weather for a city',
  parameters: { /* ... */ },
  execute: async ({ city }) => { /* ... API call logic ... */ },
};

const client = new MerciClient({ token });

// 2. Configure the agent with your tools
const agent = client
  .chat('google-chat-gemini-flash-2.5')
  .withTools([weatherTool]);

// 3. Run it! The SDK handles the entire conversation.
const finalAnswer = await agent.run("What's the weather like in Paris?");

console.log(finalAnswer);
// > The weather in Paris is sunny with a temperature of 14°C.
