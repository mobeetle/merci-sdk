Lesson 1: Hello, world!

In this lesson, you will learn how to create a simple "Hello, world!" application using the Merci SDK. This application will demonstrate the basic structure of a Merci program and how to interact with the JetBrains AI Platform.

Code

Here's the code for the "Hello, world!" application. This code should be in a file named lesson_1.mjs.

import { merci } from '@jetbrains/merci-sdk';
async function main() {
  const response = await merci.sayHello('World');
  console.log(response);
}
main();

Expected Output

When you run the lesson_1.mjs file, you should see the following output in your console:

Hello, World!
